🏷️ CSS 클래스(Class)와 아이디(ID)

HTML 태그들에게 별명을 붙여주고 싶을 때가 있어요!
그럴 때 사용하는 것이 바로 클래스(Class)와 아이디(ID)예요.

교복 같은 클래스 (Class)

👕 클래스는 "여러 명"이 같이 써요!

학교 교복이나 축구팀 유니폼을 생각해보세요.
같은 반 친구들이 모두 같은 교복을 입는 것처럼,
여러 개의 태그에 똑같은 스타일을 주고 싶을 때 사용해요.

  • 여러 번 사용할 수 있어요.
  • CSS에서는 이름 앞에 마침표(.)를 찍어서 불러요.
.우리반친구들 {
  color: green;
}
이름표 같은 아이디 (ID)

🆔 아이디는 "딱 한 명"만 써요!

우리 집 주소나 나의 주민등록번호처럼,
단 하나뿐인 특별한 태4그를 찾을 때 사용해요.
한 페이지에 똑같은 아이디를 여러 개 쓰면 컴퓨터가 헷갈려해요!

  • 딱 한 번만 사용해야 해요.
  • CSS에서는 이름 앞에 샵(#)을 찍어서 불러요.
#나만의보물상자 {
  background-color: gold;
}

🎯 이것만 기억하세요!

🏠 클래스(.) : "모두 모여라!" (여러 개 꾸미기)
☝️ 아이디(#) : "너만 나와!" (딱 하나만 꾸미기)
