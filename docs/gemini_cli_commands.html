Gemini CLI Slash Commands

Gemini CLI에서 사용할 수 있는 슬래시(/) 명령어 가이드입니다.

명령어 설명
/about 버전 정보를 표시합니다. 이슈 보고 시 유용합니다.
/agents 로컬 및 원격 하위 에이전트를 관리합니다.
(서브 명령어: list, reload, enable, disable, config)
/auth 인증 방식을 변경하기 위한 대화창을 엽니다.
/bug GitHub에 Gemini CLI 관련 이슈를 생성합니다. 입력한 텍스트가 이슈 제목이 됩니다.
/chat 또는 /resume 세션 체크포인트 및 내보내기를 관리합니다.
(서브 명령어: debug, delete, list, resume, save, share)
/clear 터미널 화면과 스크롤백을 지웁니다. (단축키: Ctrl+L)
/commands .toml 파일에서 로드된 커스텀 슬래시 명령어를 관리합니다.
(서브 명령어: list, reload)
/compress 토큰 절약을 위해 현재 대화 맥락을 요약으로 대체합니다.
/copy 마지막 출력을 시스템 클립보드에 복사합니다.
/directory (또는 /dir) 멀티 디렉토리 지원을 위해 워크스페이스 디렉토리를 관리합니다.
(서브 명령어: add, show)
/docs 웹 브라우저에서 Gemini CLI 문서를 엽니다.
/editor 지원되는 텍스트 편집기를 선택하는 대화창을 엽니다.
/extensions CLI 확장을 관리합니다.
(서브 명령어: config, disable, enable, explore, install, link, list, restart, uninstall, update)
/help (또는 /?) 사용 가능한 명령어와 사용법에 대한 도움말을 표시합니다.
/hooks 동작 커스터마이징을 위한 라이프사이클 훅을 관리합니다.
(서브 명령어: disable-all, disable, enable-all, enable, list)
/ide IDE 통합 및 컴패니언 설정을 관리합니다.
(서브 명령어: disable, enable, install, status)
/init 현재 디렉토리를 분석하고 GEMINI.md 맥락 파일을 생성합니다.
/mcp Model Context Protocol (MCP) 서버 및 도구를 관리합니다.
(서브 명령어: auth, desc, disable, enable, list, reload, schema)
/memory GEMINI.md 파일에서 로드된 명령 맥락을 관리합니다.
(서브 명령어: add, list, refresh, show)
/model 모델 설정 및 선택을 구성합니다.
(서브 명령어: manage, set)
/permissions 폴더 신뢰 설정 및 권한을 관리합니다.
(서브 명령어: trust)
/plan 현재 작업 계획을 확인하기 위해 계획 모드(읽기 전용)로 전환합니다.
(서브 명령어: copy)
/policies 활성화된 보안 및 운영 정책을 나열합니다.
(서브 명령어: list)
/privacy 개인정보 보호 고지 사항을 표시하고 데이터 수집 동의를 관리합니다.
/quit (또는 /exit) Gemini CLI를 종료합니다. --delete 옵션으로 세션 기록을 삭제할 수 있습니다.
/restore 도구 실행 전 상태로 프로젝트 파일을 복구합니다.
/rewind 대화 상태나 파일 변경 사항을 되돌리기 위해 히스토리를 거슬러 올라갑니다. (단축키: Esc 두 번)
/settings .gemini/settings.json을 수정하기 위한 인터랙티브 설정 편집기를 엽니다.
/shells (또는 /bashes) 백그라운드 쉘 보기를 토글하여 장기 실행 프로세스를 관리합니다.
/setup-github 이슈 분류 및 PR 리뷰를 위한 GitHub Actions를 구성합니다.
/skills 특수 워크플로우를 위한 에이전트 기술을 관리합니다.
(서브 명령어: disable, enable, list, reload)
/stats 세션, 모델(토큰 사용량), 도구 통계를 표시합니다.
(서브 명령어: session, model, tools)
/terminal-setup 다중 라인 입력을 위한 터미널 키바인딩을 구성합니다.
/theme CLI의 시각적 테마를 변경하기 위한 대화창을 엽니다.
/tools 사용 가능한 도구를 나열합니다.
(서브 명령어: desc (상세 설명), nodesc (이름만))
/upgrade 브라우저에서 Gemini Code Assist 업그레이드 페이지를 엽니다.
/vim 입력 프롬프트에서 Vim 스타일 탐색 및 편집 모드를 토글합니다.
