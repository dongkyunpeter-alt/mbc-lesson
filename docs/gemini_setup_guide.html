Gemini CLI 실행 가이드

윈도우 사용자를 위한 편리한 실행 환경 설정 방법

이 문서는 생성된 설정 파일들을 사용하여 윈도우 탐색기 어디서든 Gemini CLI를 즉시 실행하는 방법을 설명합니다.

방법 1 마우스 우클릭 메뉴 등록 (추천)

윈도우 탐색기에서 폴더를 우클릭하거나 빈 공간을 우클릭하여 바로 Gemini CLI를 실행할 수 있습니다.

  • 파일 실행: 폴더 내 add_gemini_to_context_menu.reg 파일을 더블 클릭합니다.
  • 권한 허용: 레지스트리 편집기 경고창이 뜨면 "예"를 클릭합니다.
  • 사용하기:
    • 임의의 폴더를 선택하고 마우스 오른쪽 버튼을 누릅니다.
    • 폴더 내부 빈 공간에서 마우스 오른쪽 버튼을 누릅니다.
    • 메뉴에서 Gemini CLI 여기서 열기를 클릭합니다.
  • 제거하기: remove_gemini_from_context_menu.reg 파일을 실행하면 메뉴가 즉시 삭제됩니다.
방법 2 배치 파일 직접 실행

명령어 입력 없이 파일 클릭만으로 Gemini CLI를 실행하는 방법입니다.

  • 파일 확인: open_gemini.bat 파일을 확인합니다.
  • 사용하기: 해당 파일을 더블 클릭하면 현재 폴더 위치에서 Gemini CLI 터미널이 열립니다.
  • 팁: 이 파일을 바탕 화면이나 자주 사용하는 폴더로 복사해두면 더욱 편리합니다.
주의 사항:

이 기능이 작동하려면 시스템에 gemini-cli가 설치되어 있어야 하며, gemini 명령어가 환경 변수(PATH)에 등록되어 있어야 합니다.

작성된 파일 목록

  • add_gemini_to_context_menu.reg: 우클릭 메뉴 등록 스크립트
  • remove_gemini_from_context_menu.reg: 우클릭 메뉴 제거 스크립트
  • open_gemini.bat: 원클릭 실행 배치 파일
